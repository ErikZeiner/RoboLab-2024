How was RoboLab built?

The main functionality was built using JavaScript. The site design was done using Tailwind with an additional package called DaisyUI and theming was achieved using CSS Theme Change. I also used code-input for IDE-like user window. Robot icon created by Freepik - Flaticon. This website was built without any AI-generated code!
